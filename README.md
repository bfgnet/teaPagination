teaPagination
=============

Esta clase para PHP te permitirá combinar de muchas maneras la paginación de datos también con AJAX.

Instalación
-----------

Puede clonar directamente el proyecto con el comando: 

git clone git://github.com/bfgnet/teaPagination.git

o descargar el archivo. Descomprimir dentro de su proyecto dejando la carpeta *Installation*.
